int sfs_create ( char *  pathname,
int  type 
)

Creates a new file or directory at the path specified.

Create a file with the pathname specified. If there is not already a file with the same pathname, the pathname must contain the full directory path. The parameter type specifies whether a regular file or a directory file should be created.

Parameters
pathnameThe pathname of file to create, it must be a full directory path.
typeThe type of file to create, zero for a regular file, one for a directory.
Returns
Returns an integer value. If the value > 0 the file creation was a success, and if the value <= 0 the file createion was unsuccessful.
Exceptions
INVALID_FILE_NAMEIf the specified file name is already in use.
INVALID_FILE_TYPEIf the file type specified is not a regular file (0) or a directory file (1).
INVALID_PATHIf the pathname specified does not already exist.
INSUFFICIENT_DISK_SPACEIf the length of the blocks to be written is greater than the amount of available blocks on disk.
